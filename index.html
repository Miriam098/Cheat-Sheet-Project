HTML5 BASIC TAGS

Standard Structure
<html> These tags are usedat the beginning and end of an HTML document. This is known as the "root element." Using these tags tells the website that the webpage is being written in HTMLS, and that all the other tags within these are HTML for­ matted.
<head> Use these to group information specific to the page. This includes title tags, meta descriptions, and links to both script and style sheets.
<title> This is used for the title tag of the page. Here you'll place a title that is useful to both search engines and users.
<body> Between these tags is where you'll place all the content for users. This includes text images, and graphic elements.
<script/> All the information or links to external scripts.
<link/> Creates association with external pages like style sheets.
<meta> Includes additional information about the page. This includes descriptions, author, publish date, keywords, and other•off-page" information.
<style> Used to include document style information which is usually CSS

HTML5 COMMON TAGS

Standard Structure
<strong> Places strong emphasis on something (displays in bold on browsers)
<em> The emphasis tag. Turns text into italics for on browsers.
<cite> This highlights references information. Good for quotes and statements.
<del> Labels a previously deleted portion of text.
<blockquote> These are used for long paragraphs of quotation which are commonly cited as well.
<br/> This creates a line break used for writing blocks of text on diffent lines
<sub> Subscript text that is smaller and placed a half-line lower than previous text.
<b> Also creates bold text, but not as effective as the tag above.
<i> Analternate way to create italics, but mainly used for thoughts or names.
<pre> Monospace text laid out with the whitespace inside the element intact.
<ins> Shows a section that has been inserted into the content
<q> Used for shorter quotations
<hr/> Create a horizontal rule or sectional break. This is u to creat a change in topic or section on the page.
<sup> Superscript text which is half a line higher than previous text
<div> Containers used to label a page section or the placement of an element
<p> Used to organize
<img> This tag is used to display image files
<table> These tags define all the content related to a table
<form> Tags used to create a form. Combined with attributes to show how the form will operate
<input/> The input attribute shows what type of field information to receive from users
<ul> Bullet point list (no particular order).
